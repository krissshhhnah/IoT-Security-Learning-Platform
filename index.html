17 OSI-Aligned IoT Attack Directory

Explore specific vulnerabilities and launch simulation nodes straight into the sandbox environment to study telemetry response and event logging.

01 DATA LINK

Man-in-the-Middle

Intercepts transit frames between gateway and sensors to manipulate payloads or spoof active commands.

02 PHYSICAL

Eavesdropping

Sniffs raw RF carrier signals over unencrypted wireless links to capture credential hashes and device configurations.

03 NETWORK

Replay Attack

Captures valid authentication tokens and broadcasts them later to bypass hardware verification schemes.

04 DATA LINK

Spoofing Attack

Forges MAC addresses of trusted IoT devices to gain illegal network entry and bypass access control lists.

05 NETWORK

Packet Injection

Crafts and injects malicious hex payloads directly into active routing queues to trigger remote shell overrides.

06 TRANSPORT

Denial of Service

Floods the local IoT gateway router with junk request frames, causing CPU overhead and memory exfiltration.

07 TRANSPORT

Distributed DoS

Inundates target router queues using multiple compromised zombie nodes, causing severe network outages.

08 PHYSICAL

Jamming Attack

Emits high-intensity RF carrier noise to flood the radio spectrum, causing 100% wireless packet drop rate.

09 APPLICATION

Credential Theft

Exfiltrates password databases and active device cryptographic handshakes from insecure local storage nodes.

10 SESSION

Session Hacking

Hijacks active BLE connection states or session keys, tricking smart devices into accepting illegal control packets.

11 PHYSICAL

Rogue Insertion

Injects unverified rogue hardware modules into active physical connections, bypassing local network firewalls.

12 NETWORK

Routing Attack

Manipulates mesh routing tables (e.g. RPL) to blackhole payloads or reroute data paths through threat nodes.

13 NETWORK

Sybil Attack

Spawns multiple pseudonymous virtual clone nodes to flood mesh routing algorithms and forge voting consensus.

14 PRESENTATION

Sensor Tampering

Injects forged telemetry data (e.g. manipulating light/temperature values) to trigger false system shutdowns.

15 PHYSICAL

Timing Attack

Uses side-channel exfiltration to monitor microcontroller crypto execution cycles and exfiltrate private keys.

16 PHYSICAL

Physical Access

Taps target board JTAG/UART headers to probe raw memory dumps or flash malicious firmware directly.

17 TRANSPORT

Delay Attack

Holds back and buffers packet delivery times to compromise control-loop synchronization and timing parameters.

Security Impact Matrix

Learn how vulnerabilities cascading across IoT devices disrupt mission-critical environments.

HIGH THREAT

Physical Infrastructure Damage

Manipulated sensor telemetry and delayed commands can trigger mechanical failure in water treatment facilities, smart generators, and assembly line robots.

Impacted by: Sensor Tampering, Delay Attack, Rogue Insertion
MEDIUM THREAT

Confidential Data Exfiltration

Unencrypted wireless communication enables RF eavesdropping, allowing malicious actors to sniff authentication keys, usernames, and sensor tracking records.

Impacted by: Eavesdropping, MITM, Credential Theft
CRITICAL THREAT

Network Failure & Spectrum Denial

Continuous RF spectrum jamming or flood-based Denial of Service renders network nodes unreachable, completely shutting down smart home monitoring queues.

Impacted by: Jamming Attack, DoS, DDoS
